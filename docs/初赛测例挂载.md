### 挂载初赛镜像文件

目前能测试初赛题目的是 25 年的，题目以磁盘镜像的形式封装了起来，想在容器中使用终端查看镜像中的文件可以使用以下方法

0. 运行脚本下载镜像文件

```bash
os/scripts/get_img.sh
```

1. 创建一个目录作为 ISO 镜像文件的挂载点（具体路径需要你自己改）

```bash
sudo mkdir os/img/mnt
```

2. 使用 mount 命令挂载 ISO 镜像文件，参数 -o loop 选项是告诉系统使用回环设备（loop device）来挂载 ISO 文件

```bash
sudo mount -o loop os/img/sdcard-rv.iso os/img/mnt
```

3. 现在，你可以像访问普通目录一样访问 mnt/ 目录来查看和操作 ISO 镜像文件中的内容

4. 卸载挂载的 ISO 文件，可以使用 umount 命令

```bash
sudo umount /mnt/iso
```
